\section{Methodology}

We introduce Mamba-3, a state space model with three new innovations:
``exponential-trapezoidal'' discretization for more expressive dynamics (\cref{sec:method:trap}),
complex-valued state spaces for state tracking (\cref{sec:method:complex}),
and multi-input multi-output (MIMO) to improve modeling power and inference-time hardware utilization (\cref{sec:method:mimo}).
These advances address the quality, capability, and efficiency limitations of current sub-quadratic architectures.
We combine these together into an updated Mamba architecture block in \cref{sec:method:arch}.
